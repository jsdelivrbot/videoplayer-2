# videoplayer

This component is built using just React. 

## To run

```
git clone https://github.com/rupshabagchi/videoplayer.git
cd videoplayer
``` 

and 

```
npm install
npm start
```
or

```
yarn
yarn run
```


## Features

* Instantaneous search (so works faster than youtube)
* Clean look
* Customizable
