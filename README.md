# Videoplayer

This component works on a browser and is built using just React.


## Features

* Instantaneous search (so works faster than youtube)
* Clean look
* Customizable

## To run

```
git clone https://github.com/rupshabagchi/videoplayer.git
cd videoplayer
```

and

```
npm install
npm start
```
or

```
yarn
yarn run
```
